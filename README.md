# IPL 2022 Match Analysis Project

This project focuses on **Exploratory Data Analysis (EDA)** of **IPL 2022 match-level data** using Python.  
The goal is to understand **team performance, toss impact, winning patterns, and match outcomes** through visualizations.

---

## Dataset Overview

The dataset contains match-wise information such as:
- Teams playing the match
- Venue and date
- Toss winner and toss decision
- First & second innings scores
- Match winner and winning method
- Player of the match and top performances

Each row represents **one IPL match**.

---

## Key Analysis Performed

- Most winning teams in IPL 2022  
- Toss decision trends (Bat vs Field)  
- Impact of toss winner on match result  
- Matches won by **Runs vs Wickets**  
- Teams converting toss wins into match wins  

---

##  Visualizations Used

- Bar plots
- Count plots
- Team-wise comparison charts

Libraries used:
- **Pandas**
- **Matplotlib**
- **Seaborn**

---

## Key Insights

- Most teams preferred **fielding first** after winning the toss  
- Majority of matches were **won by wickets**, highlighting chasing dominance  
- Toss winner had an advantage, but **performance mattered more**  
- Some teams converted toss wins into match victories more effectively
